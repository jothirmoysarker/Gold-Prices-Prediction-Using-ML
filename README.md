# Gold Price Prediction System using Machine Learning

This project focuses on predicting gold prices using multiple machine learning and time series models.  
The system analyzes historical financial data and applies advanced forecasting techniques to improve prediction accuracy.  
All model training, evaluation, and visualization were conducted in Google Colab.

## Features

- Time Series Analysis for trend and pattern detection
- Implementation of multiple ML models:
  - Convolutional Neural Networks (CNN)
  - K-Nearest Neighbors (KNN)
  - Random Forest
  - Logistic Regression
- Ensemble-based prediction technique for improved accuracy
- Data preprocessing and feature engineering
- Performance evaluation with visualizations
- Developed and executed in Google Colab

## Requirements

To run this project, you need:

- Python 3.x
- Google Colab (recommended)
- Required Python libraries:
  - numpy
  - pandas
  - matplotlib
  - scikit-learn
  - tensorflow / keras (if using ANN)
- Dataset containing gold price historical records

## How to Run the Project

Follow the steps below to execute the system:

1. Open Google Colab  
   Go to: https://colab.research.google.com/

2. Upload the notebook  
   Upload the project `Gold_Prices_Prediction_Using_Machine_Learning.ipynb` file to Google Colab.

3. Upload the dataset  
   Dataset is alrady added to the code no upload neened (you can run the url in the code to download the dataset).

4. Install required libraries  
   Run the setup cell to install missing dependencies.

5. Run the notebook  
   Execute each cell in order to preprocess the dataset, train models, and view predictions.


## Project Description

This machine learning system predicts future gold prices by combining classical and modern algorithms.  
It leverages both time series analysis and ensemble modeling to improve forecasting reliability.  
The project includes visualization tools that help compare model performance and understand prediction behavior.
